About

This page will show the information about the selected data set.

Data set is regarding a furniture retailer in The United States.

The dataset is taken from kaggle website. Click here to go to the original reference.

For a retail furniture store, predicting future sales is critical to avoiding inventory issues like overstocking or under-stocking. The challenge lies in utilizing time series data from the superstore dataset to forecast furniture sales for the next year accurately. This predictive insight ensures an optimal customer experience, avoids losses, and maintains store sustainability.

The Original Author of the Dataset is: Elzahra' Ala'


The detailed description of each variable of dataset are given below:

Sr. No. Variable Name Description
1. Row_ID Represents serial number of each entry. Datatype: INT
2. Order_ID Unique ID for each order. Datatype: VARCHAR
3. Order_Date Date on which a particlar order is placed. Datatype: DATE
4. Customer_Name Name of the Customer who placed order. Datatype: VARCHAR
5. Segment Which customer category the buyer belong to. Datatype: TEXT
6. State From which state the Customer has placed order. Datatype: TEXT
7. Postal_Code Area code of the buyer's address. Datatype: INT
8. Sub_Category Sub Category or the ordered item. Datatype: VARCHAR
9. Product_Name Name of the product ordered. Datatype: TEXT
10. Sales Total Price of the product(s) purchased. Datatype: DECIMAL
11. Quantity Quantity of the product ordered. Datatype: INT
12. Discount Discount on total purchase. Datatype: DECIMAL
13. Profit Profit earned on the purchase. Datatype: DECIMAL
